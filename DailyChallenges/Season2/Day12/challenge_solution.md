#### Adjust Priorities

1. Find PID of cpu\_hog.sh:

```bash
ps aux | grep cpu_hog
jobs -l
```

2. Lower priority:

```bash
renice 5 -p <PID>
```

3. Increase priority (requires sudo):

```bash
sudo renice -5 -p <PID>
```

4. Verify:

```bash
ps -lp <PID> > ~/processlab/priority.txt
```
